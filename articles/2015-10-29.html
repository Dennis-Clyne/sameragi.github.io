octopressを他のパソコンからdeployする



前提条件

  • deployしたいパソコンにdeployできるoctopressディレクトリがある
  • Bitbucketにoctopressのバックアップがある

本題

octopressのディレクトリにBitbucketからバックアップをpullする

        $ git pull git@bitbucket.org:UserName/octopress.git

_deployにGitHubのリポジトリからpullする

        $ cd _deploy
        $ git pull git@github.com:UserName/UserName.github.io.git

new_post["title"]等の変更を行い、generateしてdeployする
Bitbucketにpushする

        $ git add -A
        $ git commit -m "commit comment"
        $ git push
